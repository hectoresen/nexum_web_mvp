Nexum

A minimal, self-hosted voice and chat communication system

Lightweight alternative to Discord and TeamSpeak. Your data, your rules. No accounts required.

Download for Windows View on Github
Built with Rust, Tauri & React • Open Source • Privacy-First

Why Nexum?

A lightweight, elegant alternative to Discord, TeamSpeak, and Mumble

Self-Hosted

Run your own server locally or remotely. Your data, your rules. No central infrastructure required.

No Accounts

Just enter a username and connect. No email, no password, no registration hassle.

Lightweight

Built with Rust and Tauri. Efficient resource usage and modern desktop UI.

Text Chat

Real-time messaging with SQLite persistence. Create channels and chat instantly.

Voice Chat

Low-latency voice communication with UDP. Perfect for gaming and team calls.

Easy Management

Manage server from client UI. Create channels, configure settings, all in one place.

Download Nexum for your platform and start communicating
